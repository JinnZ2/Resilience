# Nutrient Cascade Model

Maps the full pathway from soil mineral depletion through crop nutrient density,
human biochemistry, cognitive/physical capacity, and civilization throughput.

## The Cascade

```
SOIL MINERALS (Zn, Se, Mg, Fe, K — declining since industrialization)
      │
      ▼
CROP NUTRIENT DENSITY (bred for yield, not density; declining per-calorie)
      │
      ▼
FOOD PROCESSING (strips remaining micronutrients)
      │
      ▼
HUMAN BIOCHEMISTRY ◄── BOTTLENECK: everything downstream rate-limited here
      │
      ├─────────────────────────────────────────┐
      │                                         │
      ▼                                         ▼
COGNITIVE CAPACITY                     PHYSICAL CAPACITY
  - problem solving                      - labor output
  - innovation rate                      - injury recovery
  - institutional trust                  - immune function
  - impulse control                      - reproductive viability
  - long-horizon planning                - circadian/sleep quality
  - social cooperation                   - chronic disease load
      │                                         │
      └─────────────┬──────────────────────────┘
                    │
                    ▼
           CIVILIZATION THROUGHPUT
             - manufacturing & innovation
             - maintenance & repair
             - reproduction & training of next generation
```

## US Baseline State (pre-shock)

Current population nutrient status from NHANES and published research:

| Nutrient | Estimated Adequacy | Impact Domain |
|----------|-------------------|---------------|
| Mg | ~50% below RDA | anxiety, impulse control, sleep, muscle function |
| K | ~97% inadequate | cardiovascular, muscle, nerve function |
| Zn | declining in grain | immune function, cognition, fertility |
| Se | regional soil depletion | thyroid, immune, antioxidant defense |
| Fe | rising insufficiency | energy, cognition, immune function |
| D | widespread deficit | immune, bone, mood regulation |
| DHA/EPA | low in standard diet | brain structure, inflammation |
| B12 | declining with age | nerve function, cognition, energy |

**Key insight**: The US population is operating at ~60-70% micronutrient
baseline BEFORE any supply shock occurs.

```
        ▼

```

## The τ_α(N_t) Extension

In the food system model, τ_α is treated as a parameter.
In reality:

```
τ_α = f(N_t, K_t, I_t)

  N_t = population nutrient status
  K_t = available practical knowledge
  I_t = institutional capacity

  All three are currently degrading.
  τ_α is not fixed. It is accelerating away from recovery.
```

This makes the food system model's "ratchet" even worse than the base
equations show, because the ratchet mechanism itself is degrading.
