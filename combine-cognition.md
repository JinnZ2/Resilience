# Combine Cognition

Probability space navigation architecture. Navigates outcome spaces
rather than optimizing for single-point predictions.

## Core Concept

```
CONVENTIONAL AI/DECISION SYSTEMS:

  State → Optimize → Best single action
  
  Problem: "best" assumes you know the loss function,
           the state space is complete, and the future
           is predictable. All three assumptions fail
           in cascading systems.


COMBINE COGNITION:

  State → Map probability space → Navigate toward
          viable outcome regions → Track consequences
          across 6 dimensions → Adjust trajectory

  Not: "what's the best move?"
  But: "where are the viable regions and how do we stay in them?"
```

## Five-Layer Architecture

1. **Probability Space Mapping** — Enumerate reachable outcome regions, not point predictions
2. **Fractal Recombination Interface** — Complementary negative space; what the map DOESN'T show is as informative as what it does
3. **Consequence Anchoring** — Track outcomes across 6 dimensions simultaneously (not reducible to single utility)
4. **Navigation Layer** — Trajectory planning through probability space toward viable regions
5. **Feedback Integration** — Continuous recalibration from observed outcomes

## Key Insight: Navigation vs. Optimization

```
OPTIMIZATION:                    NAVIGATION:
  
  single objective function        multiple viability constraints
  finds the peak                   stays in the habitable zone
  brittle to model error           robust to surprise
  assumes known landscape          explores unknown landscape
  fails catastrophically           degrades gracefully
```

This is the decision architecture applied to the corridor mapping
and food system model: not "what's the optimal policy?" but
"what keeps 280,000 people inside the viable region for 5 more years?"

## Connection to Repository

Combine Cognition provides the decision framework that sits on top of
the sensor data (corridor mapping), the dynamic models (food system,
nutrient cascade), and the institutional assessment (thermodynamic analysis).
