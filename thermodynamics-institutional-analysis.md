# Thermodynamic Institutional Analysis

Every institution is a heat engine. It has irreducible purpose, measurable efficiency,
and ungameable drift signals visible in public records.

## Core Framework

```
INSTITUTION AS HEAT ENGINE:

  INPUT ENERGY                    OUTPUT WORK
  (funding, labor,    ──►  [ INSTITUTION ]  ──►  (services, goods,
   resources, mandate)      │            │        outcomes, decisions)
                            │   WASTE    │
                            │   HEAT     │
                            ▼            │
                      (overhead, rent-   │
                       seeking, mission  │
                       drift, internal   │
                       friction)         │

  EFFICIENCY = useful output / total input

  Second law applies: efficiency < 1, always.
  But the GAP between theoretical max and actual
  tells you exactly how much energy is being
  wasted on internal friction vs. mission delivery.
```

## The Seven Modules

1. **Purpose Extraction** — Identify the irreducible function (what breaks if this institution vanishes?)
2. **Energy Budget** — Map all inputs (funding, labor, mandate authority)
3. **Output Measurement** — Measure actual delivered outcomes (not reported metrics)
4. **Waste Heat Accounting** — Quantify overhead, internal conflict, compliance theater
5. **Efficiency Calculation** — Ratio of useful work to total input
6. **Drift Detection** — Track efficiency over time; declining = institutional capture or decay
7. **Criticality Classification** — Shuttle/submarine tier framing

## Criticality Tiers

```
TIER 1: SHUTTLE/SUBMARINE
  Failure = catastrophic, irreversible loss of life
  Examples: nuclear safety, dam inspection, air traffic control
  Required efficiency: near-maximum
  Tolerance for drift: ZERO

TIER 2: INFRASTRUCTURE
  Failure = cascading degradation across systems
  Examples: power grid, water treatment, freight logistics
  Required efficiency: high
  Tolerance for drift: LOW

TIER 3: SERVICE
  Failure = reduced quality of life, recoverable
  Examples: education, healthcare administration, housing
  Required efficiency: moderate
  Tolerance for drift: MODERATE (but compounds)

TIER 4: DISCRETIONARY
  Failure = inconvenience
  Required efficiency: low bar
  Tolerance for drift: HIGH
```

## Ungameable Signals

The key insight: institutions can manipulate their reported metrics,
but they cannot hide certain thermodynamic signatures:

- **Headcount-to-output ratio over time** (available in public budgets)
- **Response latency** (measurable externally)
- **Error rate in core function** (visible in outcomes data)
- **Internal communication volume** (proxy for friction)
- **Leadership turnover pattern** (proxy for internal instability)

These signals are ungameable because they emerge from the physics of
the organization, not from its self-reporting.
